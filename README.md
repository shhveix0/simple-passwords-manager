# Simple Password Manager (Beta v0.1)

A simple and lightweight Python-based password manager for console use. It allows you to quickly save passwords to a local file and view them when needed.

## Features
- **Password Saving**: Fast recording of new data into a text-based database.
- **Database Viewing**: Access to the list of saved passwords using a code word.
- **ASCII Art Interface**: Stylish retro-hacker themed design on startup.
- **Automation**: The program automatically creates the database file upon the first run.

## Installation and Setup

1. Make sure you have [Python 3.x](https://www.python.org/) installed.
2. Download the `manager.py` file.
3. Open your terminal (or command prompt) in the folder containing the file.
4. Run the program:
   ```bash
   python manager.py
